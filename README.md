# AI Heart Disease Prediction System

## Description

This project is an Artificial Intelligence application developed in Python for cardiovascular disease prediction using clinical patient data.

The system implements a complete Machine Learning pipeline, including data preprocessing, exploratory data analysis (EDA), model training, evaluation and deployment. A Random Forest classifier was trained using the UCI Heart Disease Dataset to identify patients at risk of developing cardiovascular diseases.

To improve transparency and trustworthiness, the project integrates Explainable Artificial Intelligence (XAI) through SHAP (SHapley Additive Explanations), allowing users to understand which clinical factors contributed most to each prediction.

The trained model is deployed through a Streamlit web application that enables real-time risk assessment based on patient medical information. In addition to generating predictions, the system provides interpretable insights into the decision-making process, helping users better understand the model's output.

This project demonstrates the practical application of Machine Learning, Data Science and Explainable AI techniques in the healthcare domain.

# Sistem AI pentru Predicția Afecțiunilor Cardiovasculare (RO)

## Descriere

Acest proiect reprezintă o aplicație de Inteligență Artificială dezvoltată în Python pentru predicția afecțiunilor cardiovasculare pe baza datelor clinice ale pacienților.

Sistemul implementează un flux complet de Machine Learning, incluzând preprocesarea datelor, analiza exploratorie (EDA), antrenarea modelelor, evaluarea performanței și implementarea aplicației. Un clasificator Random Forest a fost antrenat utilizând setul de date UCI Heart Disease pentru identificarea pacienților cu risc cardiovascular.

Pentru creșterea transparenței și încrederii în predicțiile generate, proiectul integrează tehnici de Explainable Artificial Intelligence (XAI) prin intermediul bibliotecii SHAP (SHapley Additive Explanations), oferind explicații detaliate privind factorii care au influențat fiecare predicție.

Modelul antrenat este implementat într-o aplicație web realizată cu Streamlit, care permite evaluarea în timp real a riscului cardiovascular pe baza informațiilor medicale introduse de utilizator. Pe lângă rezultatul predicției, aplicația oferă și interpretarea deciziei modelului pentru o mai bună înțelegere a procesului de clasificare.

Proiectul demonstrează aplicarea practică a tehnologiilor moderne de Machine Learning, Data Science și Explainable AI în domeniul medical.
